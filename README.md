# Recipe App


Desktop application for recalculate recipe ingredients based on a specified weight of one ingredient.
The app currently works with predefined recipes and focuses on calculation logic rather than recipe management.

## Features
- View predefined recipes 
- Recalculate ingredient quantities proportionally
- Simple desktop UI

## Tech Stack
- Python
- Kivy
- Object-Oriented Programming (OOP)

## How to run
1. Clone the repository
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
   

3. Run the application:
```bash
    python main.py
